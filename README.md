This part is a beginner guide for creating repo on github

1. Initialize a Local Repository:

   1. Create a local folder for learning_php project
   2. Run git init that creates a hidden .git folder, turning the project into a Git repository.

2. Stage and Commit Files

   1. Create README.md file
   2. Run git add
   3. Run git commit -m "Initial commit

3. Connect Local Repo to GitHub

   1. Create GitHub repo and connect it to the local repo: git remote add origin "(github repo link)"

4. Rename Default Branch (from master to main)

   1. Run git branch -M main

5. Push Code to GitHub
   1. Push the local main branch to the remote repository: git push -u origin main
